## How to add screenshots

When demoing the skill live, capture screenshots of:
1. The campaign brief input
2. Claude asking the 3 clarifying questions
3. The discovery step (showing web_search queries)
4. The final Markdown report output (scorecard + DMs)

Place screenshots in this folder named: `demo-[step]-screenshot.png`
